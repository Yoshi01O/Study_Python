データ型の章

データ型一覧

文字列　　　型：文字列。
整数　　　　型：小数点を含まない数値。
浮動小数点　型：小数点を含む数値。
日付　　　　型：日付。
ブール　　　型：TrueやFalse。
配列　　　　型：複数の要素を［］で囲んだデータ。","で区切る。要素の変更不可
タプル　　　型：複数の要素を（）で囲んだデータ。","で区切る。要素の変更可能
辞書　　　　型：複数の要素を｛｝で囲んだデータ。：の前後が対応している。","で区切る。要素の変更可能
セット　　　型：複数の要素を｛｝で囲んだデータ。同一な要素が存在しない。","で区切る。

文字列(str)型


i= "文字は"
print(i)
print(type(i))

整数(int)型


i= 0
print(i)
print(type(i))

浮動小数点(float)型


i= 1.0
print(i)
print(type(i))

日付(datetime)型


import datetime
i= datetime.date.today()
print(i)
print(type(i))

ブール(bool)型


i= 0==0 
print(i)
print(type(i))

配列(list)型


i= [0,1,2]
print(i)
print(type(i))

タプル(tuple)型


i= (0,1,2)
print(i)
print(type(i))

辞書(dict)型


i= {0:"ゼロ",1:"イチ",2:"ニ"}
print(i)
print(i[0])
print(type(i))

セット(set)型


i= {0,0,1,2}
print(i)
print(type(i))
